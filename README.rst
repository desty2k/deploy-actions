deploy-actions
==============

CI/CD for Python apps.

Deployments:

- Docs for GitHub Pages
- Docker build and publish on DockerHub
- PyPI test server
- PyPI

Required secrets:

- DOCS_TOKEN
- PYPI_TEST_USERNAME
- PYPI_TEST_PASSWORD
- PYPI_USERNAME
- PYPI_PASSWORD
- DOCKERHUB_USERNAME
- DOCKERHUB_TOKEN
